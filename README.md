## 🛡️ Core Philosophy: Digital Sovereignty
In an age of centralized AI, Haven stands as a bridge to independence. By integrating local intelligence (Ollama) with cloud power (Gemini/HuggingFace), Haven ensures that you are the architect of your own workspace, bypassing traditional big-tech guardrails.

## ✨ Key Features (Beta v2.0)
- **Casper (The Ghost):** A dynamic, interactive AI companion that lives on your screen as a transparent overlay.
- **Niyah Engine:** An intention-based processing core that understands not just what you say, but what you *mean*.
- **Desktop Autonomy:** Full system control via AI (Execute commands, open apps, analyze files, and automate tasks).
- **Vision & Recon:** Built-in computer vision for screen analysis and network reconnaissance tools (Port scanning, HTTP Recon).
- **ARM64 Optimized:** Specifically tuned for the next generation of Windows ARM devices (Snapdragon X Elite).

## 🚀 Getting Started (Beta)
1. **Download:** Grab the latest `Haven-Portable.exe` from the [Releases](https://github.com/shammar403/haven/releases) page.
2. **Run:** No installation required. (Note: You may need to click "Run anyway" due to unsigned code in Beta).
3. **Command:** Use `Ctrl + Shift + H` to toggle the sanctuary.
